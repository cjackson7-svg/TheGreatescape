# Great Escapes Texas — Website Redesign
A complete redesign of the Great Escapes Texas escape room website.  
Created as part of my Web Design & Development coursework, this project focuses on improving usability, mobile responsiveness, clarity of information, and the overall user experience.

---

## Project Overview
Great Escapes Texas is an escape room company offering themed, interactive puzzle rooms for groups.  
The original website included outdated layouts, inconsistent fonts, unclear booking paths, and limited mobile optimization.

This redesign focuses on:

- Organizing room information clearly  
- Improving the visibility and consistency of booking calls-to-action  
- Updating the visual structure and layout  
- Ensuring responsive performance across devices  
- Improving readability and navigation flow  
- Enhancing user experience through spacing, hierarchy, and clarity  

The goal was to modernize the site while maintaining the brand’s adventurous and mysterious identity.

---

## Objectives
- Redesign the homepage with stronger visuals and clearer messaging  
- Improve the booking process for a more intuitive user journey  
- Provide structured layouts for room descriptions, difficulty levels, and group sizes  
- Optimize the site for both mobile and desktop users  
- Apply a consistent visual style across all pages  
- Replace outdated images with optimized, high-quality alternatives  
- Improve navigation and content architecture
